# 🚀 Estruturas de Dados I & II (Data Structures)

Este repositório reúne as implementações e estudos desenvolvidos durante as disciplinas de **Estrutura de Dados I e II** do curso de **Bacharelado em Ciência da Computação** no **Instituto Federal Catarinense (IFC)**.

O foco principal é o domínio da linguagem **C** para a construção de algoritmos eficientes, explorando desde a alocação dinâmica de memória até a organização lógica de estruturas não-lineares.

## 📂 Organização do Repositório

O projeto está estruturado em módulos independentes, facilitando a navegação pelos conceitos de cada semestre:

* **[`binary-search`](./binary-search/)**: Algoritmo de busca binária em vetores ordenados.
* **[`linked-lists`](./linked-lists/)**: Implementações de Listas Encadeadas Simples e Duplamente Encadeadas com manipulação de ponteiros.
* **[`queues`](./queues/)**: Filas Dinâmicas para organização sequencial de processos.
* **[`stacks`](./stacks/)**: Pilhas dinâmicas com interface visual em console e rotinas de validação de dados.
* **[`trees`](./trees/)**: Árvore Binária de Busca (BST) para gerenciamento hierárquico de dados.

## 💡 Conceitos Explorados no IFC

Durante o desenvolvimento, foram aplicados conceitos fundamentais da Ciência da Computação:

* **Gerenciamento de Memória:** Uso rigoroso de `malloc`, `free` e análise de *memory leaks*.
* **Ponteiros e Referências:** Manipulação direta de endereços, encadeamento de nós (`nodes`) e estruturas autorreferenciadas.
* **Recursividade:** Implementação de algoritmos recursivos para percursos e operações em árvores.
* **Análise de Complexidade:** Avaliação de performance utilizando a Notação Big O ($O(1)$, $O(n)$, $O(\log n)$).
